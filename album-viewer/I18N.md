## Usage

### In Components

Import and use the `useI18n` composable:

```vue
<script setup lang="ts">
import { useI18n } from './composables/useI18n'

const { t } = useI18n()
</script>

<template>
  <h1>{{ t.header.title }}</h1>
  <p>{{ t.header.subtitle }}</p>
</template>
```

### Available Translations

All translations are accessed through the `t` object:

```typescript
t.header.title          // Header title
t.header.subtitle       // Header subtitle
t.loading               // Loading message
t.error.message         // Error message
t.error.retryButton     // Retry button text
t.albumCard.addToCart   // Add to cart button
t.albumCard.preview     // Preview button
t.language.select       // Language selector label
t.language.en           // English language name
t.language.fr           // French language name
t.language.de           // German language name
```

## Adding New Translations

To add translations for a new text:

1. **Update the interface** in `src/i18n/translations.ts`:

```typescript

  // ... existing translations
  myNewSection: {
    myNewKey: string
  }
}
```

2. **Add translations for all languages**:


## How It Works

### Composable Pattern

The `useI18n` composable provides:
- `t` - Computed property with current translations
- `currentLanguage` - Current selected language
- `setLanguage(lang)` - Function to change language
- `availableLanguages` - Array of supported languages

### Persistence

The selected language is automatically saved to localStorage and restored on app reload.

### Reactivity

All components using `useI18n` automatically re-render when the language changes, thanks to Vue's reactivity system.

## Type Safety

TypeScript ensures:
- All translation keys exist
- No typos in translation paths
- Consistent structure across all languages
- Autocomplete support in your IDE

## Testing Different Languages

1. Click the language selector in the header
2. Choose a language from the dropdown
3. All text updates immediately
4. Reload the page - your selection is remembered

## Translation Guidelines

When adding translations:
- Keep text concise and natural
- Consider cultural context
- Maintain consistent tone across languages
- Test UI layout with longer translations (e.g., German tends to be longer)
- Use native speakers for review when possible
